Oracle Autonomous Database

Um serviço de banco de dados totalmente automatizado que facilita para todas as organizações desenvolver e implementar cargas de trabalho de aplicações, independentemente da complexidade, escala ou criticidade. O mecanismo convergente do Autonomous Database oferece suporte a diversos tipos de dados, simplificando o desenvolvimento e a implementação de aplicações, desde modelagem e codificação até ETL, otimização de banco de dados e análise de dados. Com ajuste, dimensionamento e aplicação de patches automatizados orientados por machine learning, o Autonomous Database oferece o mais alto desempenho, disponibilidade e segurança para cargas de trabalho de OLTP, análise, lote e Internet das Coisas (IoT). Construído no Oracle Database e no Oracle Exadata, o Autonomous Database está disponível na Oracle Cloud Infrastructure para implementações compartilhadas ou dedicadas e on-premises com Exadata Cloud@Customer e Dedicated Region Cloud@Customer.

Por que escolher o Autonomous Database?

Para desenvolvedores

O Oracle Autonomous Database oferece uma plataforma unificada e flexível que permite utilizar sua escolha de LLM, seja de código aberto ou proprietário, com o Select AI, que traduz linguagem natural em consultas ao banco de dados sem necessidade de codificação. Ele atende rapidamente às demandas do seu negócio ao permitir o uso de múltiplos tipos de dados — como SQL, JSON, gráficos, dados geoespaciais, texto e vetores — em um único banco de dados, eliminando a necessidade de integrar soluções separadas. Além disso, oferece opções de implantação na nuvem ou em seu data center por meio da nuvem distribuída da OCI.

Para analistas

Com o Oracle Autonomous Database, você obtém insights rápidos por meio de conversas em linguagem natural, sem operações manuais complexas, carrega e transforma dados facilmente com recursos de arrastar e soltar, dispensando o suporte do TI, e utiliza um único banco de dados para todos os formatos de dados, eliminando a necessidade de integrar e gerenciar múltiplos sistemas.

Para tecnologia da informação

O Oracle Autonomous Database automatiza tarefas administrativas como provisionamento, backups e monitoramento, permite que desenvolvedores e analistas interajam com os dados usando linguagem natural sem integrações ou operações complexas, e protege seus dados com detecção e correção automatizadas de ameaças.
